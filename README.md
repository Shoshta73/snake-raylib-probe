# Snake

A simple game of snake written in C++ using raylib.

## How to build

### Linux

#### ninja

```bash
mkdir build
cd build
cmake -G Ninja ..
ninja
```

#### makefiles

```bash
mkdir build
cd build
cmake ..
make
```

and then run the executable:

```bash
./snake
```
